# Оборудыш

Telegram Mini App для бронирования съёмочного оборудования и студии 626 Media BMSTU (МГТУ им. Н. Э. Баумана).

## Возможности

- Бронирование оборудования: даты, время, каталог, корзина и статус заявки.
- Бронирование студии 626.
- Роли пользователя, верификация и блокировки.
- Рабочие места куратора, администратора и старшего администратора.
- Выдача и возврат оборудования с чек-листом и фото.
- Уведомления, переписка по заявке, статистика, рассылки и резервные копии.

## Структура

- `prototype/` — интерфейс мини-приложения: разметка, стили и каталог.
- `prototype/texts.js` — редактируемые видимые подписи интерфейса.
- `bot/` — Python-бэкенд, Telegram-бот, база SQLite и тесты.
- `bot/texts.py` — редактируемые сообщения и пожелания Telegram-бота.
- `DEPLOY.md` — развёртывание на Windows и VPS.
- `ТЗ Оборудыш WebApp.md` — техническое задание.
- `Карта кода бота/` — карта устройства бота и гайды по развитию.

## Локальный запуск

1. Перейдите в папку `bot/`.
2. Создайте файл `.env` по примеру `.env.example` и заполните параметры.
3. Установите зависимости: `pip install -r requirements.txt`.
4. Запустите: `python main.py`.

Для локальной разработки можно использовать DEV-режим из `AGENTS.md`.

## Безопасность

В репозиторий не добавляются токены, `.env`, база SQLite, резервные копии, загруженные фото и списки людей. Не публикуйте эти данные в коммитах.

## Обновления

После изменения `bot/main.py` требуется перезапуск бота. Изменения в `prototype/` применяются как статика. Подробности — в `DEPLOY.md`.
