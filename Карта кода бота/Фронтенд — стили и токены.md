# 🎨 Фронтенд — стили и токены

Весь CSS — в `<style>` в начале `index.html` ([строки 11–251](../prototype/index.html)). Один зелёный акцент, тёмная тема в приоритете.

## Дизайн-токены (CSS-переменные)

Определены в `:root` (светлая) и `.dark` (тёмная). Всё оформление — через них, хардкод цветов не использовать.

| Токен | Тёмная | Смысл |
|---|---|---|
| `--bg` / `--bg2` | #0a0a0a / #000 | фон |
| `--card` | #121212 | карточки |
| `--text` / `--hint` | #f4f5f7 / #9a9a9a | текст / подсказки |
| `--btn` / `--btn-text` | **#43ed4e** / #000 | акцент (зелёный) |
| `--sep` | #242424 | рамки |
| `--destructive` | #ef5350 | опасное/красное |
| `--ok`/`--warn`/`--info`/`--violet`/`--teal` | — | цвета статусов |
| `--mono` | JetBrains Mono | даты/ID |

> [!info] Стиль mediabmstu.ru
> Чистый чёрный фон, кнопки-пилюли (radius 99px), чёрный текст по зелёному, Montserrat, моношрифт для дат/ID. Менять только по слову заказчика.

## Ключевые классы

- `.phone` — рамка «телефона» (в браузере); в Telegram скрыта (`body.in-tg`).
- `.card`, `.list-item`, `.pill`, `.mainbtn`, `.btn-sm` — базовые блоки.
- `.msg` / `.msg.me` / `.msg.senior` — чат (senior = красный).
- `.day`, `.slot`, `.cal` — календарь и слоты.
- `.sheet` / `.sheet-veil`, `.modal` / `.modal-veil` — шторки и модалки (в `#overlay`).

## Иконки — SVG line-art (не эмодзи)

- `ICONS` — иконки категорий; `icon(cat, sz)`.
- `UICONS` + `uic(name, sz)` — UI-иконки.
- `thumb(short, sz)` — фото позиции (`img/<short>.avif`) с фолбэком на SVG-иконку.

> [!warning] Ловушки CSS (из практики)
> - `font:` шорткат не принимает `inherit`-семейство — задавай `font-family` отдельно.
> - Мобильные оверрайды требуют префикс `.phone .класс`.
> - Блок `body.in-tg` (Telegram-режим) — **в конце** `<style>`, перебивает верхние правила.
> - Зум отключён намеренно (`gesturestart`/двойной тап/multi-touch → `preventDefault`, [строки 277–286](../prototype/index.html)) — не убирать.

## Адаптив и Telegram

- `@media (max-width:430px)` — телефон на весь экран, панель ролей внизу.
- `body.in-tg` — высота по `--tg-viewport-stable-height`, `env(safe-area-inset-bottom)` для нижних кнопок.
